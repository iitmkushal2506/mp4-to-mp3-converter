About This Project

Kush's MP4 to MP3 Converter is a web-based application that converts MP4 video files into MP3 audio files using a simple and clean interface. Users can upload locally stored video files, convert them to MP3 format, preview the audio directly in the browser, and download the converted file.

Important Note

This tool is NOT a YouTube downloader or streaming converter. It only works with MP4 files that are already downloaded on your device.

How the Conversion Works

When a user uploads an MP4 file, the backend server processes the file using FFmpeg. FFmpeg extracts the audio stream from the video file and converts it into MP3 format. The resulting audio file is then returned to the browser where it can be played or downloaded.

Technologies Used

  • Python
  • Flask Web Framework
  • FFmpeg for audio extraction
  • HTML, CSS and JavaScript
  • Gunicorn production server

Code Overview

The backend is built using Flask. When a file is uploaded, the server temporarily stores it and runs an FFmpeg command using Python's subprocess module to extract the audio stream. Once conversion finishes, the MP3 file is returned to the user.

Deployment

This application is deployed using Render. The source code is hosted on GitHub. Whenever new code is pushed to the repository, Render automatically redeploys the application so the live site stays updated.

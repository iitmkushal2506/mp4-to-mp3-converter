About This Project

Kush's MP4 to MP3 Converter is a web-based tool that allows users to convert MP4 video files into MP3 audio files. The application provides a simple interface where users can upload a video file, convert it to audio format, preview the result, and download the converted MP3 file.

Important Note

This application is NOT a YouTube downloader or streaming converter. It only works with MP4 files that are already downloaded on your device.

How the Conversion Works

When a user uploads an MP4 file, the backend server processes the file using FFmpeg. FFmpeg extracts the audio stream from the video file and converts it into MP3 format. The converted audio file is then returned to the user.

Technologies Used

  • Python
  • Flask Web Framework
  • FFmpeg for audio extraction
  • HTML, CSS and JavaScript for frontend
  • Gunicorn production server

Code Explanation

The backend of the application is written in Python using Flask. The Flask server receives the uploaded file and stores it temporarily. A subprocess call is used to run FFmpeg which extracts the audio stream. Once the conversion is completed, the server sends the MP3 file back to the browser.

Deployment

This project is deployed on Render. The code is stored on GitHub and automatically deployed whenever updates are pushed to the repository.
